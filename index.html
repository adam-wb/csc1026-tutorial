Frequently Asked Questions

Why does my CSS selector not work?

CSS selectors have a different specificity value depending on if they refer to tags, IDs or elements. This website has a handy specificity calculator. Selectors with a higher specificity take priority.

Where can I learn HTML tags, CSS properties and JavaScript?

A quick google search will often reveal numerous ways to accomplish whatever you're trying to do, but a good resource for HTML, CSS and JavaScript is W3Schools. You can also test your code using JSFiddle

What is this website made with?

This website uses both the Bootstrap and jQuery frameworks, as well as icons from Glyphicon.
